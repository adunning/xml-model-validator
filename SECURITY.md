# Security Policy

## Supported Versions

Security updates are provided for the latest release only.

## Reporting a Vulnerability

Please report security vulnerabilities through [GitHub's private vulnerability reporting](https://github.com/adunning/xml-model-validator/security/advisories/new) rather than through a public issue.

Include as much detail as possible: a description of the issue, steps to reproduce it, and the potential impact. This project is maintained on a volunteer basis, so response times may vary, but accepted vulnerabilities will be addressed in the next release.
